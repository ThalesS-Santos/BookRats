# 🗺️ Roadmap do Projeto

Este documento acompanha a evolução do BookRats, desde sua concepção até as futuras expansões planejadas.

---

## ✅ Fase 1: Fundação e Core (Concluído)
- [x] Configuração inicial do Expo 54 + React 19.
- [x] Integração com Firebase (Auth e Firestore).
- [x] Arquitetura de Slices com Zustand 5.
- [x] Sistema básico de biblioteca (Adicionar, Remover, Listar).
- [x] Cronômetro de leitura imersivo.

---

## ✅ Fase 2: Social e Echoes (Concluído)
- [x] Implementação dos **Echoes** (Rastros Literários).
- [x] Lógica de filtragem Anti-Spoiler.
- [x] Galeria 3D interativa para visualização de notas.
- [x] Sistema de Reações (Rat Claps).
- [x] Threads e respostas em anotações.

---

## ✅ Fase 3: Qualidade Industrial (Concluído - Atual)
- [x] **100% de Cobertura de Testes** em módulos core (API, Slices, Services).
- [x] Implementação de MSW para mocking de rede.
- [x] Global Error Boundary para estabilidade de UI.
- [x] Logger centralizado e seguro.
- [x] Refatoração para Atomic Design.

---

## 🚀 Fase 4: Colaboração e Expansão (Próximos Passos)
- [ ] **Clubes do Livro Pro:** Grupos com leitura conjunta sincronizada e chat por capítulos.
- [ ] **AI Insights:** Sugestões de livros baseadas no sentimento das suas anotações (Echoes).
- [ ] **Offline Mode:** Suporte a leitura e criação de notas offline com sincronização posterior.
- [ ] **Ranking de Temporadas:** Competições mensais com badges exclusivos de temporada.
- [ ] **Exportação de Notas:** Exportar seus Echoes para PDF ou Notion com design elegante.

---

## 📈 Visão de Longo Prazo
O BookRats visa ser a plataforma definitiva para o "Social Reading", transformando a leitura solitária em um evento comunitário, dinâmico e recompensador.
